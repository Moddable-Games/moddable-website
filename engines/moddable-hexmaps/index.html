Engine Open source

Moddable Hexmaps.

Hex map generation for any game. Seeded randomisation, three render styles, JSON import/export. One engine powering Nukes, Colony, Talisman, and Twilight Imperium.

SUPPORTED GAMES

Four games. One hex engine.

CAPABILITIES

Generate. Edit. Share.

Every map is deterministic from a seed. Share a 6-character code and anyone gets the same board. Edit terrain in-browser, export as JSON, import into your own tools.

RENDER STYLES

Three styles. Same data.

Switch between Classic (flat colour), Artistic (tile images), and ASCII art — all from the same map seed.
