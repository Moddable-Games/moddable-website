Dungeon Chess.

Chess reimagined as an asymmetric dungeon skirmish. Four species on modular boards with spells, XP drafting, and environmental hazards. Designed from scratch by Moddable Games.

How it plays.

THE RACES

Four factions. Zero symmetry.

Each race starts with the same piece count but plays completely differently. Names provisional pending playtest.

MOD HOOKS

Fork the dungeon. Rewrite the rules.

Dungeon Chess ships with four moddable files. All in version control.
