Endless Skies.

A hex-tile exploration game built to be modded. Three biomes, twelve faction decks, one rulebook with version control — so the community can ship expansions without asking permission.

How it plays.

MOD HOOKS

The whole rulebook is in a git repo.

Endless Skies ships with four hookable files. Fork the repo, change the file, send us a PR — or run your variant privately and never tell us.
