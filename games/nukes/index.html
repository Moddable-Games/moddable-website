Nukes.

Cold-war territory control on a hex map. Three superpowers, fifteen turns, one button that ends everything. Designed to be modded — swap the superpowers, change the fallout rules, rewrite the endgame.

How it plays.

▲ THE BUTTON

Once per game.
Never take it back.

Any superpower can press The Button on any turn. Every hex within two rings of the target is permanently fallout — no resources, no movement, no rebuilding. The presser gains 5 points and the game ends two turns early.

▲ MOD HOOKS

Change the superpower. Change the stakes.

Nukes ships with four moddable files. All in version control.
